# Java Mini Bank System

A console-based banking system built using Java.

## Features
- User Registration
- Login with 3 Attempts
- Forgot Password
- Change Password
- Deposit Money
- Withdraw Money
- Transaction History
- Logout / Exit Confirmation

## Technologies Used
- Java
- ArrayList
- Methods
- Loops
- Conditional Statements

## How to Run
1. Open project in IntelliJ IDEA / VS Code
2. Run `minibank.java`
